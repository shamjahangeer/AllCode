Holiday Schedules

The Holiday Schedules page allows the user to view and update currently scheduled plant holidays.

These records are used in the delivery scorecard system in determining workdays. It is important for each site to keep their planned holidays updated so that on-time shipments falling within a holiday season are not falsely reported as late.

Sites should report their holidays by either their entire plant, (Location Code 00) or by individual department (i.e. KL01, SWAD, etc.) but NOT by both. If both are reported, the system will interpret this as a site-wide holiday followed by a holiday for that individual department within the site.
