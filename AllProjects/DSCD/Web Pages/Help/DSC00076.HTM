Delivery Scorecard Search Criteria

The Delivery Scorecard Search Criteria screen will display various query fields based on the following selection that are available on this screen:

  • Major data views are displayed in the dropdown list.

  • Data view categories vary based on the major data view selected. By selecting a specific category within the major data view, you narrow the data parameters that will appear on the screen.

     

  • If applicable, a delivery window must be selected to aid in the determination of shipped-schedule variance.
