Controller Code

The code that identifies the person responsible for planning the production or procurement of the associated part. When using these codes while attempting to generate a report, note that they are case-sensitive.

Controllers are also known as MRP Controllers, for "M"aterials "R"equirements "P"lanning.

These codes can be looked up by using the appropriate button next to their respective field in the search criteria window.

Note: Controller (scheduler, planner) codes are assigned at the Inventory Organization level or at a SAP plant level.
