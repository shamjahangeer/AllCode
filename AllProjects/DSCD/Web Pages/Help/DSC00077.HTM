Data Views

Major Delivery Scorecard data views provide you with specific, accurate data to track delivery performance. Major data views are as follows:

  • Plant/Building/Location

  • Organization
  • Customer/Worldwide
  • Shipping Facility
  • Team

  • MRP Controller

  • Make Stock
  • Product Code/Product Line
  • Industry Code
  • Manufacturing Building
  • Profit Center
  • SAP Sales Office/Group

     

To select a data view:

Click the desired selection from the dropdown list.
