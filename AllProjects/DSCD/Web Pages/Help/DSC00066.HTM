Worldwide Account Number List

The Delivery Scorecard Worldwide Account Number List window displays information for all Worldwide Customers associated with a Ship To Customer Account Number.

To access worldwide account number data

  1. Type data in the following required fields:

    Accounting Org Id
     
    Customer Number
     
     

  2. Press the Submit Button to generate the data listing.
