Customer Account Type

Three specific parameters (listed below) which filter the delivery details by the selected customer type.

If no selection is made, the system will use the default which is set to ALL.

Customer Account Type

 

Customer Account Type Description

 

Intracompany

Displays only intracompany (Tyco Electronics organization to Tyco
Electronics organization) product delivery details.

 

Trade

Displays only external Tyco Electronics product delivery details.

 

All

Displays both internal (intracompany) and external product delivery details.
