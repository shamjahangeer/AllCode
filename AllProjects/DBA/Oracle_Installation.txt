Create groups and users

groupadd dba
mkdir -p /export/home
useradd -g dba -G dba -m -d /export/home/oracle -s /usr/bin/bash oracle
passwd oracle


if oracle user is not working then
change the passwd file as shown below

vi /etc/passwd
